# History

`docs/history` 是 OpenFons 的统一历史层。

这里集中存放已经退出当前主线、但仍需要保留以便回看背景、验证边界、原始输入或历史决策依据的文档。

当前统一收口了四类历史材料：
- `docs/history/context`
  旧上下文与整理稿归档
- `docs/history/plan`
  旧规划方案、演进背景与迁移材料
- `docs/history/plan1`
  第一阶段执行与验证记录
- `docs/history/references`
  原始输入、参考材料与外部整理稿

阅读原则：
- 如果你要找当前正式口径，优先看 `docs/sot/**`。
- 如果你要找当前正在施工的内容，优先看 `docs/workbench/**`。
- 只有在需要追溯背景、核对历史判断或回看原始输入时，再进入 `docs/history/**`。
