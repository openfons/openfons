// User prompt builder
export function buildSummaryPrompt(messages: readonly TranscriptMessage[]): string {
  // Take at most 30 messages (15 user + 15 assistant alternating) to stay well
  // within Haiku's context window and keep API costs minimal.
  const sample = messages.slice(0, 30)
  const lines = sample.map(m => `[${m.role.toUpperCase()}]: ${m.text.slice(0, 500)}`)
  return `Summarize this coding session transcript:\n\n${lines.join('\n\n')}`
}

// API call shape
const response = await client.messages.create({
  model: options?.model ?? DEFAULT_SUMMARY_MODEL,
  max_tokens: 200,
  system: SYSTEM_PROMPT,
  messages: [{ role: 'user', content: buildSummaryPrompt(messages) }],
})
const summary = response.content[0]?.type === 'text' ? response.content[0].text.trim() : ''

- The SDK import is a dynamic await import('@anthropic-ai/sdk') so the module loads only when needed and does not break builds where the SDK is not installed yet.

## 9. Gotchas and Edge Cases

### Manifest loading
Memory/06_transcripts/manifests/ is partitioned by client/year/month/. Use fast-glob pattern Memory/06_transcripts/manifests/*/.json — the same glob package is already a dependency. Sort by started_at for stable output ordering.

### Clean markdown path resolution
manifest.repo_clean_path is a POSIX-relative path (e.g. Memory/06_transcripts/clean/...). Use join(root, manifest.repo_clean_path) to resolve it. Some manifests may have repo_mirror_enabled: false and an empty repo_clean_path — in that case fall back to manifest.global_clean_path. Always guard with existsSync and return '' on missing files.

### --no-ai Commander.js flag gotcha
Commander treats --no-<name> as a boolean negation flag. The option declared as .option('--no-ai', ...) will populate opts.ai as false when passed and true by default. The noAi property must be !opts.ai. This matches the existing --no-<x> patterns in Commander.

### @anthropic-ai/sdk optional dependency
The SDK import must be dynamic (await import('@anthropic-ai/sdk')), placed inside a try/catch that returns '' on MODULE_NOT_FOUND. This way the package is truly optional — the build still works even if the user hasn't installed it. Add to package.json as a regular dependency (not optional), but the code must defensively handle its absence for the period before npm install is re-run.

### HTML escaping
All user-generated content (titles, session IDs, message text, file paths) that is interpolated into HTML strings must be HTML-escaped. Write a small escHtml(s: string): string utility in render/layout.ts that handles & < > " '. This is the most common XSS source in template-style HTML generation.

### File system on Windows
The project runs on Windows (confirmed by the platform info). All path construction must use node:path join(), not string concatenation with /. The toPosixPath utility from src/utils/path.ts is already available for converting before writing into HTML href attributes.

### Large transcript sets
buildReport must not hold all parsed transcript content in memory simultaneously. Process transcripts in a streaming fashion: parse → summarize → render → write → discard. Keep only manifests (small) in memory throughout. The summary concurrency limit (5 parallel) also limits peak memory.

### .gitignore for Memory/07_reports/
The requirements specify the output dir should not be committed. The cmd-report.ts or build.ts should check if Memory/07_reports is already in .gitignore and append it if not. Alternatively document this as a manual step, since the init command scaffolds .gitignore already — the safer approach is to add Memory/07_reports/ to the default gitignore template in src/project/scaffold.ts.

### Wiki-link regex
The [[session-id]] pattern must use a non-greedy match: /\[\[([^\]]+?)\]\]/g. Session stems can contain hyphens and underscores but not ]]. The regex must not match across newlines.

### Empty / single-data-point charts
- Doughnut with all zeros: render an empty gray ring, no percentage labels.
- Line chart with 1 data point: render a single dot (circle), not a path.
- Heatmap with no data: render all gray cells (still useful as a date grid).
- Bar chart with no tools: render the SVG with a "No tool events recorded" text node.

### serve command port conflicts
Wrap the HTTP server creation in a try/catch for EADDRINUSE. Print a clear message: "Port NNNN is in use. Try --port MMMM." and exit with code 1.

### Report cache invalidation
The cache key is raw_sha256 (the SHA256 of the raw source file). This means the cache entry is permanently valid as long as the source file has not changed — which is the correct invariant. No TTL is needed.
