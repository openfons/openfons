    88→## Commands
    89→
    90→### `openmnemo init`
    91→
    92→Initialize a MemoryTree workspace in a repository.
    93→
    94→```bash
    95→openmnemo init --root <path> [options]
    96→
    97→Options:
    98→  --root <path>           Repository root (default: .)
    99→  --project-name <name>   Project name shown in reports
   100→  --goal-summary <text>   Initial goal description
   101→  --locale <locale>       Language: auto, en, or zh-cn (default: auto)
   102→  --force                 Overwrite existing generated files
   103→```
   104→
   105→### `openmnemo discover`
   106→
   107→Scan your home directory for AI transcripts and import new ones.
   108→
   109→```bash
   110→openmnemo discover --root <path> [options]
   111→
   112→Options:
   113→  --root <path>           Repository root (default: .)
   114→  --client <client>       Filter by client: all, codex, claude, gemini (default: all)
   115→  --scope <scope>         current-project or all-projects (default: all-projects)
   116→  --limit <n>             Import at most N transcripts
   117→  --format <format>       Output: text or json (default: text)
   118→```
   119→
   120→### `openmnemo import`
   121→
   122→Import a single transcript file.
   123→
   124→```bash
   125→openmnemo import --source <path> --root <path> [options]
   126→
   127→Options:
   128→  --source <path>         Path to the transcript file (required)
   129→  --root <path>           Repository root (default: .)
   130→  --client <client>       Override auto-detection: codex, claude, gemini, doubao
   131→```
   132→
   133→### `openmnemo report build`
   134→
   135→Generate a static HTML dashboard from imported transcripts.
   136→
   137→```bash
   138→openmnemo report build --root <path> [options]
   139→
   140→Options:
   141→  --root <path>           Repository root (default: .)
   142→  --output <path>         Output directory (default: ./Memory/07_reports)
   143→  --no-ai                 Skip AI summaries (no API key needed)
   144→  --model <model>         AI model for summaries (default: claude-haiku-4-5-20251001)
   145→  --locale <locale>       Report language: en or zh-CN (default: en)
   146→```
   147→
   148→### `openmnemo report serve`
   149→
   150→Serve the built report locally over HTTP.
   151→
   152→```bash
   153→openmnemo report serve --dir ./Memory/07_reports --port 10086
   154→```
   155→
   156→Then open `http://localhost:10086` in your browser.
   157→
   158→### `openmnemo recall`
   159→
   160→Show the most recent session summary (useful in AI agent hooks).
   161→
   162→```bash
   163→openmnemo recall --root . --project-name "my-project"

   165→
   166→### `openmnemo search`
   167→
   168→Full-text search over all imported transcripts (searches title, cwd, and branch).
   169→
   170→```bash
   171→openmnemo search --query "authentication bug"
   172→```
   173→
   174→Options:
   175→```
   176→--query <text>          Search query (required). Multiple words = AND match.
   177→--global-root <path>    Override global transcript root

   179→--format <format>       Output: text or json (default: text)

   181→
   182→> **Note:** The query is split into terms by whitespace; all terms must appear in the document (AND logic). Example: `--query "auth bug"` matches sessions where both `auth` and `bug` appear in the title, cwd, or branch.

   211→## Configuration
   212→
   213→Global settings are stored in `~/.memorytree/config.toml`:
   214→
   215→```toml
   216→heartbeat_interval = "30m"   # How often the daemon syncs
   217→auto_push = false            # Push to Git remote after each sync
   218→log_level = "info"           # debug | info | warn | error
   219→generate_report = false      # Auto-rebuild HTML report on each sync
   220→locale = "en"                # en or zh-CN
   221→```
   222→
   223→To enable auto-push and auto-report on every sync cycle:
   224→
   225→```toml
   226→auto_push = true
   227→generate_report = true
   228→```
