- clients: set[str] | None — if None, use all CLIENTS = {"codex", "claude", "gemini"}

Glob Patterns by client:

{
    "codex": ["sessions/**/*.jsonl"],
    "claude": ["projects/**/*.jsonl"],
    "gemini": [
        "tmp/*/checkpoints/**/*.json",
        "tmp/*/checkpoints/**/*.jsonl",
        "history/**/*.json",
        "history/**/*.jsonl",
        "chats/**/*.json",
        "chats/**/*.jsonl",
    ],
}

Algorithm:
1. Iterate clients in sorted order
2. Get root from default_client_roots; skip if missing/nonexistent
3. For each pattern, glob recursively
4. Filter to files only (path.is_file())
5. Dedup by resolved posix path lowercased
6. Append tuples (client, path.resolve())
7. Sort by (safe_file_mtime(path), path.as_posix().lower()) descending (newest first)
8. Return sorted list

Returns: list[tuple[str, Path]] of (client, absolute_path) sorted newest first

infer_project_slug(parsed) -> str

Algorithm (try in order):
1. If parsed.cwd exists as Path:
- Return slugify(cwd_path.name)
2. If parsed.client == "claude":
- Get parent directory name
- Strip regex r"^[a-z]--" prefix (lowercased, e.g., "c--" → "")
- Return slugify(stripped_name)
3. If parsed.client == "gemini" and source_path parent name exists:
- Return slugify(source_path.parent.name)
4. Default: "unknown-project"

transcript_matches_repo(parsed, repo_root, repo_slug) -> bool

Algorithm:
1. Resolve repo_root to absolute path
2. If parsed.cwd is valid Path:
- If cwd_path == repo_root: return True
- If repo_root in cwd_path.parents: return True
3. Fallback: return project_slugs_match(infer_project_slug(parsed), repo_slug)

project_slugs_match(left, right) -> bool
- Returns: bool(left and right and left == right and left != "unknown-project")
- Requires: both non-empty, equal, and not "unknown-project"

safe_file_mtime(path) -> float
- Try path.stat().st_mtime
- On OSError: return 0.0

### 4. COMMON HELPERS (_TRANSCRIPT_COMMON.PY)

Key functions used by above modules:

slugify(value, fallback="session") -> str
- ASCII-encode (drop non-ASCII)
- Lowercase
- Replace runs of non-alphanumeric/dash/dot/underscore with single dash
- Strip leading/trailing dash/dot/underscore
- Return fallback if empty result

sha256_file(path) -> str
- Read file in 1MB chunks, compute SHA256
- Return 64-char lowercase hex string

normalize_timestamp(value, fallback=None) -> str
- Input: int/float (Unix timestamp), datetime, or str (ISO8601)
- Output: ISO8601 string with Z suffix (UTC), no microseconds
- Handles: Z suffix conversion to +00:00, tzinfo conversion to UTC
- Fallback: current UTC time if unparseable

timestamp_partition(timestamp) -> tuple[str, str]
- Parse ISO8601 timestamp
- Return (year_str, month_str) from datetime.strftime("%Y"), strftime("%m")
- On parse error: return ("unknown", "unknown")

yaml_escape(value) -> str
- Escape: \ → \\, " → \", newline → \n
- Wrap in double quotes: f'"{escaped}"'

load_json(path) -> dict[str, Any] | None
- If path doesn't exist: return None
- Read UTF-8, parse JSON
- On OSError/JSONDecodeError: return None

_content_hash(text) -> str
- SHA256 of text.encode()
- Return first 16 chars of hex digest

deduplicate_messages(messages) -> list[TranscriptMessage]
- Dedup by (timestamp, role, _content_hash(text))
- Keep first occurrence, discard duplicates

deduplicate_tool_events(events) -> list[TranscriptToolEvent]
- Dedup by (timestamp, _content_hash(summary))
- Keep first occurrence

### 6. CRITICAL PATH LOGIC

Import Flow (for TypeScript implementation):
1. Parse transcript → ParsedTranscript object
2. Call import_transcript() with parsed + paths
3. Compute SHA256 of source file (first 8 chars for artifact stem)
4. Partition timestamp → year/month tokens
5. Build 5 paths (repo raw/clean/manifest, global raw/clean/manifest, event log, DB)
6. Create directories
7. Copy source file
8. Build manifest dict (15 keys total)
9. Load existing manifest from global_manifest_path
10. Preserve import timestamp if content unchanged
11. Write clean markdown (YAML frontmatter format)
12. Write manifest JSON (pretty-printed, 2-space indent)
13. Append to event log ONLY if manifest is new or changed
14. Upsert SQLite (4-tuple PK, 14 data columns)
15. Return result with global_index_db added

Discovery Flow:
1. Enumerate clients (sorted)
2. For each client, glob patterns from client root
3. Deduplicate by resolved lowercase posix path
4. Sort by mtime descending (newest first)
5. Return [(client, path), ...]

Project Slug Inference:
1. Try cwd directory name
2. Try source file parent (Claude: strip [a-z]-- prefix)
3. Default to "unknown-project"

Matching Logic:
- Repo matches transcript if: cwd == repo_root OR repo_root in cwd.parents OR project_slugs_match
- Slugs match if: both non-empty, equal, and not "unknown-project"
